Orders manual

    In this screen you can view all of the orders that have been made by customers from the first order till the most recent one. The table shows the buyer's info as well as the products that they purchased along with the price of each order.

  1. Filter table

    By using the filters above the table headers, you can shrink the data in the table and filter the data according to your preferences.

  2. Search bar

    The search bar is also another way of filtering table data. Use it when you want to find a specific user.

  3. Table data

    You can click on the left side of the screen to filter how many entries you want to display in every page of the table.

  4. Delete Order

    The delete button will delete from the system all of the information corresponding to that specific order number. No other orders will be affected.
