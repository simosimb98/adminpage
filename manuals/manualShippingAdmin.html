Admins shipping details manual

    In this screen you can view all of the administrators shipping information.

  1. Filter table

    By using the filters above the table headers, you can shrink the data in the table and filter the data according to your preferences.

  2. Search bar

    The search bar is also another way of filtering table data. Use it when you want to find a specific user.

  3. Table data

    You can click on the left side of the screen to filter how many entries you want to display in every page of the table.

  4. Edit Button

    By clicking the edit button on a certain administrator, a modal will pop up with all the current information. Validation of the data will happen once you click "Save Changes". If the new details you entered are correct, then a succes message will show. If not, a red text will pop up right under the field in which the information entered is not correct addressing the error.
