Newsletter manual

    In this screen you can view all of the newsletter subscribers in the table. You can also delete a certain subscriber and also send a newsletter from this page to all the subscribers

  1. Filter table

    By using the filters above the table headers, you can shrink the data in the table and filter the data according to your preferences.

  2. Search bar

    The search bar is also another way of filtering table data. Use it when you want to find a specific user.

  3. Table data

    You can click on the left side of the screen to filter how many entries you want to display in every page of the table.

  4. Send newsletter button

    Click this button when you are ready to send a newsletter. A modal will appear with three fields. All of them are mandatory except the file attachment button. When you click the send newsletter button the information you entered will be checked. If everything is correct a success message will appear. Otherwise a red text will appear right under the field in which the error is.

  5. Delete Button

    The delete button will delete the subsciber and that person will no longer receive any newsletters.
