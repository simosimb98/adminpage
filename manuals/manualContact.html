Contact Messages manual

    In this screen you can view all of the messages sent by users from the contact form in the main website.

  1. Filter table

    By using the filters above the table headers, you can shrink the data in the table and filter the data according to your preferences.

  2. Search bar

    The search bar is also another way of filtering table data. Use it when you want to find a specific user.

  3. Table data

    You can click on the left side of the screen to filter how many entries you want to display in every page of the table.

  4. Delete Button

    By clicking the delete button on a certain message, all the information corresponding to that message will be deleted from the system. The email will remain in the inbox of the company's email.
