Dashboard User Manual

In this screen you can view statistics about the entire system. You can view specific information from orders, messages, users products and so on. Below there's a more detailed description about each end everey single component of this module, starting from left to right.

    .
  1. Sales ammount(Current Month)

    This box displays the total ammount of the current month. It automatically updates with each and every sale happening during that month. It allways resets back to 0 at the start of a new month.

  2. Total Messages

    This number displays the total ammount of messages that exist in the database. You can navigate to the messages window to display all the messages that have not been deleted.

  3. Total Users

    This numbers displays the total ammount of users that have been activated by the administrators.

  4. Pending requests

    This number displays the total ammount of users that have not been activated yet. You can navigate to users/pending users to view and activate these users.

  5. Line chart

    This chart displays and compares the ammount of sales within a three month period. The Y axis shows the money and the X axis shows the months. Hovering on the dotted lines can display the exact ammount in sales for that month.

  6. Top sellers

    This table displays the top 3 sellers of all time. Top sellers are determined by the quantity of products that they sell and not by the total ammount of money that they make.
